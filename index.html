About

Chatbot.js is a lightweight framework that allows for easy building and deployment of chatbots for a variety of tasks. Chatbot.js comes with built in string matching and easy to use combinational logic and random reply matching. We are proud to provide chatbot.js under the MIT Open Source License.

Documentation

Chatbot.js is simple once you understand its structure. A chatbot is comprised of different states, replies for those specific states, and transitions to change between differing states.

State: The chatbot is a state machine. This means it can facilitate different responses for different contexts. For example you might want a state that can provide jokes while another state is for providing contact information.

Replies: Replies are the responses that the bot will reply to for a given keyword. They can have multiple necessary keywords and multiple replies(that will be selected at random).

Transitions: In order to change state, we provide transitions which change when a given keyword is entered.

Example Code:

Create and initialize a chatbot:

var bot=new bot("name");

Get the bot's response:

return=bot.reply(input);

Create a state for the chatbot:

bot.addstate("state");

Add a reply:

bot.addreply("state",["keyword"],["reply"]);

Add a reply with a transition:

bot.addreply("previousstate",["keyword"],["reply"],"newstate");

Add multiple replies:

bot.addreply("state",["keyword"],["reply1","reply2"]);
//picked at random

Add multiple required keywords:

bot.addreply("state",["keyword1","keyword2"],["reply"]);
//both are required

Set a state's default reply:

bot.addreply("state",["default"],["reply"]);

Set the reply for an unknown keyword:

bot.addreply("state",["n/a"],["reply"]);
